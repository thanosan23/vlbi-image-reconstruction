Abstract

Radio interferometry arrays—such as the VLA, ALMA, and the EHT—are composed of multiple radio telescopes operating in synchrony. These telescopes, often separated by vast distances, work together to observe astronomical radio sources with a resolution far beyond what a single dish could achieve. Each pair of telescopes samples a specific spatial frequency: closer pairs detect broad, smooth features, while distant pairs capture finer details. The Event Horizon Telescope (EHT), for example, is a global-scale array that leverages the entire Earth as a virtual dish. Since the placement of telescopes directly influences image resolution and quality, optimizing their configuration is essential. This project introduces an interactive visualization tool that simulates the image formation process in interferometric arrays. It enables scientists and educators to experiment with different configurations—ground-based or satellite-based—without the cost or constraints of real-world deployment. By visualizing UV coverage and image reconstruction outcomes, this tool supports both informed scientific decision-making and broader educational engagement.

Background & Motivation

The EHT’s groundbreaking image of a black hole was the result of decades of work and over $60 million in funding. However, the addition of each new telescope requires substantial investment, both financially and logistically. The scientific benefit of a new telescope depends heavily on its position relative to the existing array, making efficient placement critical. Beyond cost, geopolitical factors can limit where telescopes can be built.

Some regions restrict such installations due to political boundaries or lack of infrastructure, introducing further constraints in array planning. Given these challenges, there is a clear need for a system that allows scientists to simulate and visualize how new configurations will impact image quality before making costly decisions. This tool addresses that need, offering a way to explore telescope placement, satellite involvement, and signal processing strategies virtually. By enabling this kind of preemptive analysis, the tool can help guide the development of next-generation interferometers with greater efficiency and lower risk.

Introduction

Interferometry is a technique in radio astronomy that combines the signals from multiple radio antennas to form a single, high-resolution image of a distant source. Each antenna in an array observes the same celestial object from a slightly different angle. By correlating these signals, the array achieves a resolution equivalent to that of a telescope the size of the maximum distance between the antennas—known as the baseline. This technique allows astronomers to resolve structures that would otherwise be impossible to distinguish with individual telescopes. Arrays like ALMA and the VLA utilize this method on a continental scale, while the EHT operates globally. Some components of the EHT, such as ALMA itself, are interferometers on their own, consisting of dozens of antennas that work together as a local array before contributing to the larger network.

One of the main challenges in radio interferometry is that it does not capture an image directly. Instead, it samples spatial frequency components of the sky, which are then processed to reconstruct the original image. Due to incomplete sampling—since no array can cover all possible baselines—this reconstruction is imperfect and often includes noise or artifacts. To address this, algorithms like CLEAN are used. CLEAN iteratively identifies the strongest signal points (or "peaks") in the observed image and subtracts them, helping to reveal the underlying source structure more clearly. In advanced imaging, the ehtim library is also employed to simulate and reconstruct images with realistic constraints.

Our Tool

Our tool allows users to simulate the image produced by a radio interferometry array with a high degree of flexibility and interactivity. Users can upload a custom model image—such as a black hole simulation or a collection of point sources—and adjust various array parameters including wavelength and observation duration. Telescope positions can be added, removed, or moved freely, including the option to incorporate satellite-based telescopes. Satellite-specific parameters such as eccentricity and arc perigee are also customizable. Additionally, users can modify the source direction and fine-tune the parameters of the CLEAN algorithm, such as loop gain and threshold, to see how these affect the reconstructed image.

The tool provides multiple outputs to help interpret the simulated configuration. These include the beam pattern, the Fourier transform of the input image, the UV coverage corresponding to the array layout, and the resulting dirty image. This dirty image is then processed using both the CLEAN algorithm and the ehtim pipeline to produce refined reconstructions. Through this simulation and visualization workflow, the tool enables deeper insight into how different configurations influence final image quality.
